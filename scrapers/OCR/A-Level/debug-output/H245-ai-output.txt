2.3.7.4 7.07d Be able to use the terms “basic feasible solution”, “basic variable” and “non-basic variable” appropriately.
2.3.7.5 7.07e Understand what an iteration of the simplex algorithm achieves, in terms of moving along edges of a multi-dimensional convex polygon.
2.3.7.6 7.07f Be able to explain, algebraically, some of the calculations used in the simplex algorithm.
2.3.8 Game theory
2.3.8.1 7.08a Understand the idea of a zero-sum game and its representation by means of a pay-off matrix.
2.3.8.2 7.08b Be able to reduce a matrix using a dominance argument.
2.3.8.3 7.08c Be able to identify play-safe strategies and stable solutions and understand what they represent.
2.3.8.4 7.08d Be able to identify a Nash Equilibrium solution and understand what it represents.
2.3.8.5 7.08e Be able determine an optimal mixed strategy for a game with no stable solution by reducing to two variables and using simultaneous equations or a graphical method.
2.3.8.6 7.08f Be able to determine an optimal mixed strategy for a game with no stable solution by reformulating the problem as a linear programming problem that could be solved using the simplex algorithm.

2.4 Additional pure mathematics
2.4.1 Sequences and series

2.4.1.3 8.01c Understand and be able to describe various possibilities for the behaviour of sequences.
2.4.1.4 8.01d Identify and be able to use the limit of the nth term of a sequence as n approaches infinity, including steady-states.

2.4.1.9 8.01i Be able to extend their knowledge of modelling to second order recurrence relations.

2.4.2.1 8.02a Understand and be able to work with numbers written in base n.

2.4.2.9 8.02i Understand the concepts of prime numbers, composite numbers, highest common factors, and coprimality.
2.4.2.10 8.02j Know and be able to apply the result that a|b and a|c implies a|(bx + cy) for any integers x and y.

2.4.2.12 8.02l Know and be able to use Fermat’s little theorem.
2.4.2.13 8.02m Know and be able to use the fact that p-1 is not necessarily the least positive integer n for which a^n ≡ 1 (mod p).

2.4.2.15 8.02o Be able to use the binomial theorem to show that (a + b)^p ≡ a^p + b^p (mod p) for prime p, and use this result.
2.4.3 Groups
2.4.3.1 8.03a Be able to work with binary operations and their properties when defined on given sets.

2.4.3.4 8.03d Recognise and be able to use the Latin square property for group tables.
2.4.3.5 8.03e Recall the meaning of the term “order”, as applied both to groups and to elements of a group, and be able to determine the orders of elements in a given group.
2.4.3.6 8.03f Understand and be able to use the definition of a subgroup, find subgroups and show that given subsets are, or are not, proper subgroups.
2.4.3.7 8.03g Recall the meaning of the term “cyclic” as applied to groups.

2.4.3.12 8.03l Be able to determine whether two given groups are, or are not, isomorphic using informal methods.
2.4.3.13 8.03m Be able to work with groups defined by their algebraic properties.
